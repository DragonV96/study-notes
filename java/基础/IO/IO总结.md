# IO 总结

## 1 IO 流

### 1.1 概念

**1. I/O 概念**

I/O 是机器获取和交换信息的主要渠道。

**2. 流**

1）概念

在计算机中，流是一种信息的转换。

> 流是完成 I/O 操作的主要方式。

2）分类

- 输入流（InputStream）：机器或者应用程序接收外界的信息
- 输出流（OutputStream）：从机器或者应用程序向外输出的信息

合称为输入 / 输出流（I/O Streams）。

**3. IO 原理**

机器间或程序间在进行信息交换或者数据交换时，先将对象或数据转换为某种形式的流，再通过流的传输，到达指定机器或程序后，再将流转换为对象数据。
